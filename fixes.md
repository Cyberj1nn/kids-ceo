# Корректировки после первого запуска

## ~~1. Socket: "Невалидный токен"~~ ✅ ИСПРАВЛЕНО

~~**Проблема:** Socket.io не получает JWT-токен при подключении — клиент не передаёт `accessToken` в `handshake.auth`.~~  
**Причина:** `VITE_WS_URL` не был задан, сокет подключался к `localhost:3000`. Теперь WS URL выводится из `VITE_API_URL`.  
**Коммит:** `5a5a522`

## ~~2. Чат: нет авторефреша~~ ✅ ИСПРАВЛЕНО (следствие #1)

~~**Проблема:** Новые сообщения видны только после F5.~~  
**Причина:** Следствие проблемы #1. После починки сокета `message:new` события работают в реальном времени.

---

## ~~3. Лекции / Инструкции и другие контентные вкладки — зависают на "Загрузка..."~~ ✅ ИСПРАВЛЕНО

~~**Проблема:** Страницы с подкатегориями не загружают контент, вечный спиннер.~~  
**Причина:** Роуты были `path="lectures"` без именованного параметра `:tab`, `useParams().tab` возвращал `undefined`, страницы зависали. Исправлено через `useLocation().pathname`.  
**Коммит:** `5a5a522`
